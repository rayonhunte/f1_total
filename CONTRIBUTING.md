Thanks for contributing to F1 Total.

- The project name, branding, and identity remain owned by the project owner.

## How to Contribute

1. Fork the repo and create a feature branch.
2. Make focused changes and include tests/checks where relevant.
3. Run:
   - `npm run typecheck`
   - `npm run build`
   - `npm run functions:build`
4. Open a pull request with:
   - what changed
   - why it changed
   - screenshots for UI changes

## Code Quality

- Keep changes small and reviewable.
- Follow existing TypeScript/React patterns.
- Never commit secrets or private credentials.
